Mighty Heroes Arena (2023)

  • Tech Stack: Unity, Firebase, Photon, C#
  • Github URL: Project Link
Mighty Heroes Arena is a game created to test the final project for the degree, as explained in the section on other projects. Developed in Unity, it's a 3D online shooter featuring 3 vs 3 gameplay which includes a complex matchmaking system and a Firebase database integration. My contribution to this project was creating the player's movement and ttack mechanicsa, as well as designing its interface. In terms of networking, I also developed both the game server and matchmaking server and the database in Firebase.
