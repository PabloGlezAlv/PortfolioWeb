Magnet Partners

  • Category: Games
  • Tech Stack: Unity, C#, Unity Netcode for GameObjects
  • Type: Cooperative Multiplayer

Magnet Partners is an evolved and expanded version of the concept originally explored in "Magnet Sky". This project is a fully networked cooperative multiplayer game built with Unity.

The project currently focuses on core gameplay. The basic mechanics and puzzle-solving systems are fully implemented and functional over the network. visual assets, which are planned for future stages.

To accelerate development and overcome technical hurdles, I have utilized AI tools to solve complex algorithmic problems and gain a deeper understanding of Unity Netcode for GameObjects.
