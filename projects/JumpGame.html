Jump Game (CURRENTLY)

  • Tech Stack: Unity, C#, Firebase, AdMob
Jump Game is a casual mobile game, the game revolves around the player tapping one side of the screen to make the character ascend, with the objective of achieving the highest possible score. Depending on the side tapped, the character jumps in that direction. The game features a double jump mechanic, and the player can also grab onto walls, which vary in design. Throughout the map, there are coins that players can collect to purchase skins. Additionally, various events unfold as the game progresses. I have handled all aspects of the game's development, excluding the art. enemy AI.
