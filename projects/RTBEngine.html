RTBEngine - Game Engine

  • Tech Stack: C++17, OpenGL 4.3+, SDL2, Bullet Physics, FMOD, Lua, ImGui
  • Platform: Windows x64

RTBEngine is a custom 3D game engine designed for Windows, built from the ground up to understand the inner workings of modern engine architecture. It features a robust Entity Component System (ECS), a dedicated rendering pipeline, and an integrated visual editor for scene management.

The engine is currently in development, focusing on providing a clean and efficient framework for 3D game creation, including physics simulation, audio spatialization, and cross-system scripting via Lua.
